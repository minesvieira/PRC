    • Abstract

    O objetivo deste trabalho de casa foi a criação de uma ontologia para um arquivo musical. Comecei por verificar a estrutura que pretendia para a definição da ontologia pretendida, pelo que criei 3 classes no Protege: Obra, Instrumento e Partitura. De seguida, defini as relações no Protege entre as classes correspondentes: uma Obra contém um Intrumento, um Instrumento é contido por uma Obra (relação inversa), uma Partitura define um Instrumento e um Instrumento é definido por uma Partitura. Já com as classes e relações criadas, criamos uma stylesheet em XSL para transformação dos dados no formato necessário para todas as Obras que continham Instrumentos e respetivas Partituras, criando assim as classes individuais de cada um. Por fim, carreguei o ficheiro ttl para o GraphDB de modo a conseguir responder a 4 queries que o professor propôs na aula. Nota: as queries encontram-se na pasta "queries" dentro da diretoria do TPC2.
