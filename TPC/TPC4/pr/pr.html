    • Abstract

    O objetivo deste trabalho de casa foi a criação de uma aplicação com interface web sobre um Arquivo Musical, através do repositório do GaphDB. Foi criada uma ontologia para os arquivos musicais. Quando acedemos à interface web, temos uma lista de obras com vários campos sobre essa mesma obra (id, título e nº de partituras). Pretende-se que em cada obra, o utilizador possa ver as partituras dessa obra e as informações sobre a mesma. Para isso, foi criado um botão em cada obra "Ver Mais". Para isso realizei um pedido GET '/' que obtém todos os dados relativos à obra. Seguidamente, um pedido GET '/id' ao clicar no botão "Ver Mais", vai abrir a página com as informações da partitura referentes a essa obra, nomeadamente o path, id, clave e voz.
