    • Abstract

    O objetivo deste trabalho de casa foi a criação de uma queries em SPARQL para com esses dados, povoar uma ontologia sobre cinema. Essas queries terão como resultado informações sobre os filmes, atores, compositores, escritores, línguas e países. Primeiramente, comecei por executar as queries na DBPedia e guardar o resultado no formato JSON. De seguida, criei um ficheiro em python que lê o ficheiro JSON, de cada uma das classes. Seguidamente, converti os dados JSON, individualmente, para o formato TTL turtlecom base no ficheiro criado na última aula cinema.ttl. Resultaram assim 6 ficheiros, que depois juntei na secção dos "Individuals" do ficheiro inicial que fizemos durante a aula para desta forma, povoar toda a ontologia com os dados obtidos.
