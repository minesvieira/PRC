    • Abstract

    O objetivo deste trabalho de casa foi a criação de uma query em SPARQL para gerar uma base de dados sobre nadadores, ou seja, construir um dataset a partir da dbPedia. Os campos que foram pedidos pelo docentes foram os seguintes: name, gender, birthDate, birthPlace, deathDate e abstract@en. Os campos birthPlace, deathDate e abstract são campos opcionais, ou seja, estes campos podem não estar preenchidos.
