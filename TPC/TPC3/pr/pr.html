    • Abstract

    O objetivo deste trabalho de casa foi a criação de uma interface web para a execução de queries SPARQL. Esta interface web tinha como base o utilizador selecionar o repositório pretendido (repositórios do GraphDB), e escrever a query a executar, mostrando assim os resultados da mesma. O utilizador poderá também apagar a textarea com os resultados obtidos da query. Comecei por elaborar os routes com um GET que vai buscar todos os repositórios disponíveis, para serem mostrados na interface em forma de lista e um POST onde vai buscar os dados do resultados da queries, mostrados na textarea através do repositório selecionado também pelo utilizador. Tentei também implementar o botão "Apagar" para limpar o conteúdo da textarea, mas tive alguns problemas com a realização desta ação AJAX.
